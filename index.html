Politique de confidentialite - Datafree

Date de derniere mise a jour : 8 janvier 2026

1. Qui sommes-nous ?

L'application Datafree est developpee et publiee par Ydaoulat (ydaoulat@free.fr).

2. Donnees collectees

Aucune donnee personnelle sensible n'est collectee de maniere permanente.

  • Donnees techniques (anonymes) : IP, systeme d'exploitation, version de l'app (pour diagnostics).
  • Feedback volontaire : message + email (optionnel), plateforme, version d'app (envoye a Formspree).
  • Meteo locale : coordonnees GPS (temporaires, pour API publique Open-Meteo, non stockees).
  • Favoris/preferences : stockes uniquement localement sur l'appareil.

3. Services tiers utilises

  • Formspree (feedback) : Politique Formspree
  • Open-Meteo (meteo) : Politique Open-Meteo
  • data.gouv.fr (open data) : donnees publiques francaises.
  • archive.org : contenus publics d'archives.
  • Google Play Services : Politique Google

4. Utilisation des donnees

Les donnees sont utilisees uniquement pour :

  • Fournir les fonctionnalites de l'application.
  • Repondre aux feedbacks envoyes.
  • Ameliorer l'application (diagnostics techniques).

Aucune donnee n'est vendue ni partagee avec des tiers a des fins commerciales.

5. Vos droits

Vous pouvez :

  • Desinstaller l'app (supprime toutes les donnees locales).
  • Nous contacter pour suppression de vos feedbacks : ydaoulat@free.fr.

6. Enfants

L'application n'est pas destinee aux enfants de moins de 13 ans.

7. Modifications

Cette politique peut etre mise a jour. Les changements seront publies sur cette page.
